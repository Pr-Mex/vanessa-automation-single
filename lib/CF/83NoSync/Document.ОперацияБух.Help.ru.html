Документ доступен из меню: «Первичные операции» → «Операции, введенные вручную».
Документ предназначен для ручной корректировки данных учета по плану ЕПС в соответствующих учетных регистрах. Можно вносить корректирующие записи в регистры накопления и регистры сведений. Список доступных регистров открывается по кнопке «Выбор регистров». При этом в табличной части становится доступна вкладка, соответствующая выбранному регистру.
