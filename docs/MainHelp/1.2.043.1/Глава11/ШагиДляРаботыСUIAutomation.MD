##### [Ссылка на видео](https://vkvideo.ru/video-235354700_456239357)

001. Привет! В этом уроке я расскажу тебе про то, как использовать шаги, работающие с механизмом UI Automation. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00000_63902997590160_ШагиДляРаботыСUIAutomation.png)

002. Загрузим тестовый пример.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00005_63902997601872_ШагиДляРаботыСUIAutomation.png)

003. Важный момент. Механизм UI Automation - это технология компании Microsoft. Поэтому она работает только под операционной системой Windows.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00006_63902997601920_ШагиДляРаботыСUIAutomation.png)

004. Чтобы почитать про эту технологию подробнее перейди по этой ссылке.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00009_63902997603603_ШагиДляРаботыСUIAutomation.png)

005. Если коротко - то эта технология позволяет взаимодействовать с любым элементом формы любого приложения в Windows.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00012_63902997607742_ШагиДляРаботыСUIAutomation.png)

006. Например, чтобы нажать на кнопку с помощью этой технологии нужно выполнить шаг - И я делаю клик по элементу формы UI Automation

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00015_63902997609264_ШагиДляРаботыСUIAutomation.png)

007. Давай рассмотрим параметры этого шага.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00018_63902997613394_ШагиДляРаботыСUIAutomation.png)

008. Первый параметр - это идентификатор процесса.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00021_63902997615025_ШагиДляРаботыСUIAutomation.png)

009. Если нужно нажать на кнопку в стороннем приложении, то идентификатор процесса нужно будет получить с помощью внешней компоненты VanessaExt.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00024_63902997619146_ШагиДляРаботыСUIAutomation.png)

010. Если нужно нажать на кнопку в клиенте тестирования, тогда нужно получить в переменную идентификатор процесса клиента тестирования как показано в этом примере.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00028_63902997623062_ШагиДляРаботыСUIAutomation.png)

011. Движемся далее.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00031_63902997627302_ШагиДляРаботыСUIAutomation.png)

012. Второй параметр шага - это заголовок элемента.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00034_63902997628962_ШагиДляРаботыСUIAutomation.png)

013. Чтобы узнать заголовок элемента лучше воспользоваться инспектором объектов UI Automation.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00037_63902997633091_ШагиДляРаботыСUIAutomation.png)

014. Инспектор объектов находится примерно по этому пути

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00040_63902997634716_ШагиДляРаботыСUIAutomation.png)

015. Важный момент. Инспектор объектов не входит по умолчанию в начальную установку Windows.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00043_63902997637841_ШагиДляРаботыСUIAutomation.png)

016. Чтобы он появился нужно сначала установить Windows SDK.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00046_63902997639364_ШагиДляРаботыСUIAutomation.png)

017. Движемся далее.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00049_63902997643503_ШагиДляРаботыСUIAutomation.png)

018. Чтобы узнать тип элемента для третьего параметра надо также воспользоваться инспектором объектов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава11/images/00052_63902997645141_ШагиДляРаботыСUIAutomation.png)
