001. Привет! В этом уроке я расскажу тебе про контекстное меню редактора и как заменить шаг проверки элемента формы по имени на шаг проверки по заголовку и наоборот. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00000_63902997786597_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

002. Загрузим тестовый пример.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00005_63902997798016_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

003. Допустим после записи действий пользователя получился такой шаг

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00008_63902997799477_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

004. Этот шаг будет выполнять поиск элемента формы по его внутреннему имени

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00013_63902997805007_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

005. Может возникнуть задача, чтобы заменить шаг, на другой, который будет искать элемент формы не по имени, а по заголовку.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00016_63902997809039_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

006. Для этого в контекстном меню редактора надо выбрать данный пункт меню

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00022_63902997812784_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

007. Также эту команду можно вызвать с помощью комбинации клавиш контрол плюс джей

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00025_63902997817014_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

008. После выполнения команды шаг будет заменён на шаг, который ищет тот же элемент формы по заголовку

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00037_63902997823506_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

009. Повторный вызов команды заменит шаг обратно.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00040_63902997827635_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)

010. Важный момент. Для работы команды нужно, чтобы в клиенте тестирования было активно окно, которое содержит элемент формы, который используется в шаге.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава13/images/00041_63902997827847_КонтекстноеМенюРедактораЗаменитьШагПроверкиПоИмениНаШагПроверкиПоЗаголовку.png)
