003. Переменная может передаваться как значение параметра в шаге.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00005_63902994847253_ИспользованиеПеременныхВнутриПараметровШагов.png)

004. Также переменная может передаваться как часть значения параметра в шаге.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00006_63902994847538_ИспользованиеПеременныхВнутриПараметровШагов.png)

005. Данный шаг создаёт одну переменную ИмяКонтрагента.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00009_63902994849174_ИспользованиеПеременныхВнутриПараметровШагов.png)

006. Далее значение этой переменной передаётся во второй параметр этого шага

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00014_63902994854613_ИспользованиеПеременныхВнутриПараметровШагов.png)

007. Теперь рассмотрим более сложный пример. Данный шаг создаёт первую переменную.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00019_63902994860105_ИспользованиеПеременныхВнутриПараметровШагов.png)

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00024_63902994865736_ИспользованиеПеременныхВнутриПараметровШагов.png)

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00029_63902994871497_ИспользованиеПеременныхВнутриПараметровШагов.png)
