003. И выполним его

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00007_63902995168587_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00008_63902995169703_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00009_63902995169930_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

006. Ещё значение переменной можно вывести в окно сообщений пользователю с помощью специального шага. Я вывожу значение переменной.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00012_63902995171676_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

007. Также существует специальный инструмент - Табло переменных. Он находится тут. Горячая клавиша контрол. Плюс альт. Плюс дабл ю.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00018_63902995177175_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)



![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00027_63902995184536_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

008. Этот инструмент позволяет посмотреть значения любых переменных. Также он позволяет изменить их значения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00039_63902995192379_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)

009. Также если активизировать строку в табло переменных в этом поле можно увидеть готовый шаг сценария, который проверяет значение этой переменной

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава06/images/00054_63902995199925_КакПосмотретьЗначениеПеременнойТаблоПеременных.png)
