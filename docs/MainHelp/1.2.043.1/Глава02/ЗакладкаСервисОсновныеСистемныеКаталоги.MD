001. Привет! В этом уроке я расскажу тебе про то, какие бывают настройки связанные с системными каталогами. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902989543411_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

002. Данные настройки находится тут.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00007_63902989554879_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

003. В данном поле указывается каталог проекта.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00012_63902989560773_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

004. Это важная настройка. Лучше всегда указывать каталог проекта.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00015_63902989565257_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

005. В этом каталоге обычно находятся файлы, необходимые для запуска тестов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00016_63902989565664_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

006. Часто в сценариях может происходить обращение к файлам из каталога проекта.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00017_63902989566070_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

007. Второе поле - это каталог инструментов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00020_63902989567909_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

008. Это каталог, в котором находится обработка Vanessa Automation или Vanessa Automation Single.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00023_63902989572369_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

009. Важно, чтобы это поле было заполнено корректно.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00024_63902989572776_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

010. Далее идёт опция, позволяющая настроить как будет выглядеть фича файл при создании с помощью этой кнопки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00027_63902989574819_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

011. При установке этого флага появляется возможность указать файл шаблона нового фича файла.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00032_63902989580719_ЗакладкаСервисОсновныеСистемныеКаталоги.png)



![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00040_63902989589496_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

012. Продолжим. В этом поле можно указать путь к каталогу временных файлов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00045_63902989595420_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

013. Если указано значение настройки - тогда временные файлы скриптов будут создаваться в нём.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00050_63902989601336_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

014. Если значение настройки не указано, тогда будет использоваться временный каталог пользователя

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00053_63902989605809_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

015. Настройка нужна, т.к. Vanessa Automation может создавать bat файлы для запуска системных команд, а в некоторых операционных системах может быть установлен запрет на запуск таких файлов из временного каталога пользователя.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00054_63902989606219_ЗакладкаСервисОсновныеСистемныеКаталоги.png)

016. На этом всё, переходи к следующему уроку интерактивной справки.
