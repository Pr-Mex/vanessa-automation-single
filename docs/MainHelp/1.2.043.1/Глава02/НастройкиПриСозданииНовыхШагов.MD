![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00016_63902992622285_НастройкиПриСозданииНовыхШагов.png)

013. Эта опция может быть полезна, если на данном ПК не срабатывает генерация epf файлов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00031_63902992636154_НастройкиПриСозданииНовыхШагов.png)

014. Далее идёт поле - версия платформы для генерации epf.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00034_63902992637971_НастройкиПриСозданииНовыхШагов.png)

015. В этом поле указывается путь к каталогу, в котором находится установленная платформа 1С, которая будет использоваться при генерации epf файлов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00037_63902992642441_НастройкиПриСозданииНовыхШагов.png)

016. Важный момент. Для генерация epf файлов необходимо соблюдение нескольких условий.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00038_63902992642847_НастройкиПриСозданииНовыхШагов.png)

017. Первое - необходимо установить толстый клиент платформы 1С.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00039_63902992643259_НастройкиПриСозданииНовыхШагов.png)

018. Второе - на данном ПК должны запускаться файловые базы. Т.е. не должно быть проблем с выдачей лицензии сеансу 1С, запущенному в файловой базе.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00040_63902992643666_НастройкиПриСозданииНовыхШагов.png)
