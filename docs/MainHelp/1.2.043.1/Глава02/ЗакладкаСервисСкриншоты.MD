##### [Ссылка на видео](https://vkvideo.ru/video-235354700_456239261)

001. Привет! В этом уроке я расскажу тебе про настройки снятия скриншотов. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902990606789_ЗакладкаСервисСкриншоты.png)

002. Эти настройки находятся тут.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00007_63902990618226_ЗакладкаСервисСкриншоты.png)

003. Если данный флаг установлен - то при возникновении ошибки во время выполнения сценариев будут сниматься скриншоты.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00012_63902990624147_ЗакладкаСервисСкриншоты.png)

004. Также скриншоты будут сниматься если перед шагом указан специальный тег screenshot.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00015_63902990628605_ЗакладкаСервисСкриншоты.png)

005. Данную опцию рекомендуется устанавливать при запуске тестов в CI контуре.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00018_63902990630430_ЗакладкаСервисСкриншоты.png)

006. Продолжим. Если данный флаг установлен - то при возникновении ошибки будут сниматься скриншоты со всех запущенных сеансов 1С.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00023_63902990636332_ЗакладкаСервисСкриншоты.png)

007. Данную опцию также рекомендуется устанавливать при запуске тестов в CI контуре.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00026_63902990640800_ЗакладкаСервисСкриншоты.png)

008. Продолжим. Если данный флаг установлен - то скриншоты будут сниматься с помощью внешней компоненты VanessaExt.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00029_63902990642634_ЗакладкаСервисСкриншоты.png)

009. Продолжим. Данный флаг управляет снятием скриншотов, когда во время выполнения сценария был подключен браузер.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00040_63902990651621_ЗакладкаСервисСкриншоты.png)

010. Данная настройка будет работать только тогда, когда браузер был запущен с ключом отладки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00043_63902990656118_ЗакладкаСервисСкриншоты.png)

011. Продолжим. В данном поле указывается каталог, в который будут сохраняться скриншоты.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00046_63902990657953_ЗакладкаСервисСкриншоты.png)

012. Если поле не заполнено - то скриншоты сниматься не будут.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00049_63902990662422_ЗакладкаСервисСкриншоты.png)

013. Продолжим. В данном поле указываются имена процессов через точку с запятой, для которых надо снять скриншот.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00052_63902990664243_ЗакладкаСервисСкриншоты.png)

014. Имя процесса можно указывать не полностью, используя символ *.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00055_63902990668715_ЗакладкаСервисСкриншоты.png)

015. В данной настройке не нужно указывать имена процессов 1С, а нужно указывать имена процессов приложений, которые дополнительно запускаются во время тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00058_63902990670548_ЗакладкаСервисСкриншоты.png)

016. Например имя процесса эмулятора мобильного приложения. Или имя процесса текстового редактора word.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00061_63902990674029_ЗакладкаСервисСкриншоты.png)

017. Настройка имеет смысл только скриншоты снимаются с помощью внешней компоненты VanessaExt.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00064_63902990675850_ЗакладкаСервисСкриншоты.png)

018. Продолжим. В данном поле выбирается как именно будут сниматься скриншоты с помощью внешней компоненты с окон 1С.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00069_63902990681739_ЗакладкаСервисСкриншоты.png)

019. Рекомендуется устанавливать значение - все окна клиента тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00075_63902990688316_ЗакладкаСервисСкриншоты.png)

020. Продолжим. В данном поле указывается команда снятия скриншотов с помощью дополнительной утилиты.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00078_63902990691121_ЗакладкаСервисСкриншоты.png)

021. Данная настройка имеет смысл только если скриншоты будут сниматься с помощью дополнительной утилиты, а не с помощью внешней компоненты VanessaExt.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00081_63902990695586_ЗакладкаСервисСкриншоты.png)

022. На этом всё, переходи к следующему уроку интерактивной справки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00082_63902990695992_ЗакладкаСервисСкриншоты.png)
