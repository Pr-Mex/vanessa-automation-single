001. Привет! В этом уроке я расскажу тебе про настройки формирования отчета в формате СППР. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902990472637_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

002. Эти настройки находятся тут.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00009_63902990485459_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

003. Чтобы отчет формировался нужно установить этот флаг.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00014_63902990491344_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

004. В данном поле нужно указать каталог, в котором будут формироваться файлы отчета

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00022_63902990500098_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

005. Продолжим. В данном поле нужно указать имя тестируемой конфигурации. Это значение потом будет использоваться при загрузке данных ошибки в СППР.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00027_63902990505987_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

006. Если значение не указано, то будет взято имя конфигурации из сеанса менеджера тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00032_63902990511917_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

007. Продолжим. В данном поле нужно указать версию тестируемой конфигурации. Это значение потом будет использоваться при загрузке данных ошибки в СППР.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00037_63902990517794_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

008. Если значение не указано, то будет взята версия конфигурации из сеанса менеджера тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00042_63902990523694_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)

009. На этом всё, переходи к следующему уроку интерактивной справки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00045_63902990528172_ЗакладкаСервисОтчетыОЗапускеСценариевСППР.png)
