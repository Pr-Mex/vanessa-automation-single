001. Привет! В этом уроке я расскажу тебе про то, как устроена таблица клиентов тестирования. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902990715344_КакУстроенаТаблицаКлиентовТестированияВводная.png)

002. Отлично. Смотри. Эти настройки находится на этой закладке.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00004_63902990723750_КакУстроенаТаблицаКлиентовТестированияВводная.png)

003. Здесь находится таблица, в которой задаются настройки для подключения клиентов тестирования

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00011_63902990731265_КакУстроенаТаблицаКлиентовТестированияВводная.png)

004. Каждая строка таблицы это набор настроек для подключения конкретного клиента тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00013_63902990734922_КакУстроенаТаблицаКлиентовТестированияВводная.png)

005. Данную таблицу можно заполнить либо вручную, либо во время выполнения сценария, либо передать в json файле при запуске тестов из командной строки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00014_63902990735333_КакУстроенаТаблицаКлиентовТестированияВводная.png)

006. Первой строкой в таблице всегда будет специальная служебная строка, которая соответствует той базе, в которой запущена Vanessa Automation.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00015_63902990735739_КакУстроенаТаблицаКлиентовТестированияВводная.png)

007. У этой строки есть специальная пометка. У других строк такой пометки не будет.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00018_63902990737769_КакУстроенаТаблицаКлиентовТестированияВводная.png)
