##### [Ссылка на видео](https://vkvideo.ru/video-235354700_456239214)

001. Привет! Это вторая часть рассказа про кнопки, которые управляют созданием, сохранением и запуском сценариев. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902991414056_КнопкиЗапускаСценариевЧастьДва.png)

002. Далее идут две кнопки, которыые позволяют сохранить фича файл на диск.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00003_63902991422245_КнопкиЗапускаСценариевЧастьДва.png)

003. Эта кнопка просто сохраняет изменения в текущем фича файле

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00008_63902991428350_КнопкиЗапускаСценариевЧастьДва.png)

004. А эта кнопка вызывает диалог сохранения файла, чтобы сохранить фича файл с новым именем.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00013_63902991434236_КнопкиЗапускаСценариевЧастьДва.png)

005. Далее идёт кнопка, которая позволяет перечитать фича файл с диска.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00018_63902991440133_КнопкиЗапускаСценариевЧастьДва.png)

006. Далее идёт кнопка, которая позволяет запустить все сценарии из текущего фича файла на выполнение.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00023_63902991446025_КнопкиЗапускаСценариевЧастьДва.png)

007. Эта кнопка останавливает выполнение сценариев.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00028_63902991452124_КнопкиЗапускаСценариевЧастьДва.png)

008. А эта кнопка выполняет сразу два действия. Она перечитывает фича файл с диска и запускает сценарии на выполнение.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00034_63902991459237_КнопкиЗапускаСценариевЧастьДва.png)

009. И наконец эта кнопка позволяет перейти в режим дерева. При этом в дереве будет загружен текущий фича файл.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00039_63902991465331_КнопкиЗапускаСценариевЧастьДва.png)

010. На этом всё, переходи к следующему уроку интерактивной справки.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00042_63902991470475_КнопкиЗапускаСценариевЧастьДва.png)
