001. Привет! В этом уроке я расскажу тебе про то, какие бывают настройки связанные с дымовыми тестами. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902989229000_ЗакладкаСервисОсновныеДымовыеТесты.png)

002. Данные настройки находится тут.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00007_63902989240517_ЗакладкаСервисОсновныеДымовыеТесты.png)

003. Тут задаётся язык, на котором будет происходить генерация фича файлов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00012_63902989246426_ЗакладкаСервисОсновныеДымовыеТесты.png)

004. В данном поле указывается каталог, в который будут записаны итоговые feature-файлы дымовых тестов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00017_63902989252332_ЗакладкаСервисОсновныеДымовыеТесты.png)

005. В данном поле указывается каталог, в котором хранятся текстовые файлы с объектами-исключениями.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00022_63902989258237_ЗакладкаСервисОсновныеДымовыеТесты.png)

006. В данном поле указывается путь к файлу настройки сценариев.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00027_63902989264129_ЗакладкаСервисОсновныеДымовыеТесты.png)

007. При включении данного флага перед формированием feature-файлов в каталоге выходных файлов будут удалены все существующие feature-файлы

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00032_63902989270050_ЗакладкаСервисОсновныеДымовыеТесты.png)

008. При включении данного флага в формировании feature-файлов расширенных действий будут участвовать только объекты, для которых в текущей информационной базе существует хотя бы один не помеченный на удаление элемент.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00037_63902989275958_ЗакладкаСервисОсновныеДымовыеТесты.png)

009. Если этот флаг установлен, в feature файлы будут включены только те объекты, которые имеют хотя бы один элемент в текущей базе данных, не помеченный на удаление.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00042_63902989281857_ЗакладкаСервисОсновныеДымовыеТесты.png)

010. В данном поле указывается Имя конфигурации поставщика для сравнения конфигураций.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00047_63902989287757_ЗакладкаСервисОсновныеДымовыеТесты.png)
