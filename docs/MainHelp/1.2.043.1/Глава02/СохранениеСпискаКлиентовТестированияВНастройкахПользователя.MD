001. Привет! В этом уроке я расскажу тебе про то, как сохранить список клиентов тестирования в настройках пользователя. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00000_63902992924286_СохранениеСпискаКлиентовТестированияВНастройкахПользователя.png)

002. Для этого существует специальная команда.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00005_63902992934274_СохранениеСпискаКлиентовТестированияВНастройкахПользователя.png)

003. При выполнении команды настройки будут сохранены для текущего пользователя в текущей базе данных.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00009_63902992939958_СохранениеСпискаКлиентовТестированияВНастройкахПользователя.png)

004. Важный момент. При сохранении настроек Vanessa Automation в настройках пользователя, список клиентов тестирвания не сохраняется. Его надо сохранять отдельно.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава02/images/00015_63902992943836_СохранениеСпискаКлиентовТестированияВНастройкахПользователя.png)

005. Но список клиентов тестирования можно выгрузить в json файл вместе с остальными настройкми Vanessa Automation
