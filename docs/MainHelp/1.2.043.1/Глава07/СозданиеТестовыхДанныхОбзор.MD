001. Привет! В этом уроке я расскажу тебе основные моменты о создании тестовых данных. Давай откроем второй экземпляр Ванессы в режиме обучения.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00000_63902995663610_СозданиеТестовыхДанныхОбзор.png)

002. Часто при создании новых сценариев нужно наполнять базу тестовыми данными.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00001_63902995670304_СозданиеТестовыхДанныхОбзор.png)

003. Это могут быть контрагенты, товары, организации и так далее.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00002_63902995670679_СозданиеТестовыхДанныхОбзор.png)

004. Для того, чтобы быстро создавать тестовые данные во время выполнения сценариев - в Vanessa Automation существует специальный инструмент - Подготовка и загрузка данных.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00008_63902995674754_СозданиеТестовыхДанныхОбзор.png)

005. Этот инструмент позволяет генерировать специальне шаги, которые потом будут создавать тестовые данные.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00013_63902995680468_СозданиеТестовыхДанныхОбзор.png)

006. Чтобы сгенерировать тестовые шаги нужно пометить нужные метаданные и нажать на кнопку генерации фичи.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00014_63902995680875_СозданиеТестовыхДанныхОбзор.png)

007. Важный момент. Шаги создания тестовых данных работают на стороне менеджера тестирования.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00015_63902995681273_СозданиеТестовыхДанныхОбзор.png)

008. Ещё один важный момент. Удаление созданных тестовых данных автоматически не производится.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00016_63902995681679_СозданиеТестовыхДанныхОбзор.png)

009. Это нужно учитывать, если в одной базе запускается много сценариев, так как тестовые данные одного сценария могут повлиять на работу другого сценария.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00018_63902995682526_СозданиеТестовыхДанныхОбзор.png)

010. Также есть альтернативный инструмент для создания тестовых данных - Сериализатор макетов.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00024_63902995687175_СозданиеТестовыхДанныхОбзор.png)

011. Этот инструмент работает похожим образом, но создаёт специальные макеты, в которых хранятся тестовые данные.

![](https://vanessa-files.do.bit-erp.ru/Doc/1.2.043.1/MD/Глава07/images/00029_63902995692715_СозданиеТестовыхДанныхОбзор.png)
