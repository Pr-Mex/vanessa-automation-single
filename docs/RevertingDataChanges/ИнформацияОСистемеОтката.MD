Для работы необходимо установить расишрение "Откат изменений данных" 
не ниже версии 1.0.0.4 ссылка для скачивания : https://github.com/Loogosh/Reverting-Data-Changes-in-1C

Данное расишрение позволяет создавать закладки с набором данных в регистре системы отката, 
и с возможностью в любой момент времени загрузить нужный набор данных по имени закладки.
